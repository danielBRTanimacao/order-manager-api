API ENCOMENDAS

API pública para o gerenciamento de encomendas

Sobre a api

Uma solução prática e eficiente para gerenciar pedidos de clientes de forma automatizada em um sistema. Essa API pode ser usada para criar, atualizar, consultar e gerenciar informações sobre encomendas, oferecendo flexibilidade para integração com diferentes plataformas, como aplicativos móveis, sistemas de e-commerce ou dashboards administrativos.

API fornece endpoints para

  • Criar encomendas: Permite que novos pedidos sejam registrados com detalhes como cliente, produto, quantidade, preço e status.
  • Atualizar status: Oferece a possibilidade de alterar o status da encomenda, como "Processando", "Enviado" ou "Entregue".
  • Consultar pedidos: Retorna uma lista de encomendas com filtros opcionais, como cliente, status ou período.
  • Excluir ou cancelar encomendas: Gerencia cancelamentos ou exclusões de forma segura.

Como usar

  • GET /api/orders/ – Liste todas as encomendas ou filtre por status, cliente, ou intervalo de datas.
  • POST /api/orders/ – Crie uma nova encomenda enviando os dados no corpo da requisição (em JSON).
  • PUT /api/order/{id}/ – Atualize os detalhes ou o status de uma encomenda existente.
  • DELETE /api/order/{id}/ – Exclua ou cancele uma encomenda específica.

SERVIÇOS DISPONIVEIS API

  • Encomendas: GET http://127.0.0.1:8000/api/orders
  • Limite encomendas: GET http://127.0.0.1:8000/api/orders/?q={number}

GERENCIAR URLS DISPONIVEIS API

  • Adicionar encomenda: POST http://127.0.0.1:8000/api/orders
  • Update encomenda: PUT http://127.0.0.1:8000/api/order/{id}
  • Excluir encomenda: DELETE http://127.0.0.1:8000/api/order/{id}
